# Proyecto 1: Algoritmos de sincronización de tiempo

Implementación de los algoritmos de Berkeley & Cristian para sincronización de relojes.

### Ejecutar algortimo de Berkeley

Para nodo maestro

`python main.py m`

Para nodo esclavo

`python main.py e`


### Ejecutar algortimo de Cristian

Para nodo servidor

`python main.py s`

Para nodo cliente

`python main.py c`
